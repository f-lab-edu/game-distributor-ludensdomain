# ludensdomain
게임을 위한 공간. 게임 유통과 구매 및 정보 교환장

<br>

## 프로젝트 목표
- 특정 날(대규모 세일 주간)에 대용량 트래픽을 견뎌내는 게임 유통 사이트의 신비함을 구현해 보고자 함
- 단순한 기능 구현보다 의미 있고 가독성 높은 코드를 작성하고자 함(코드 몽키 탈출!)
- 기존에 사용하던 기술들을 정확히 이해하며 객체지향 정신을 받아 코드에 녹여내고자 함
